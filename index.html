mime-type icons

Site that provides mime-type icons.

How to use

You can use a file extension like /png, a filename like /excel.xls, a mimetype like /application/pdf .

You can also use the ?size=32 to get a consistent size of icons, and the ?default= to force a default icon if not found the file extension.

Examples

http://mimetypeicons-wgenial.rhcloud.com/.docx?size=16

https://mimetypeicons-wgenial.rhcloud.com/page.html?size=16

https://mimetypeicons-wgenial.rhcloud.com/image.jpeg?size=32

https://mimetypeicons-wgenial.rhcloud.com/file.doc?size=16

http://mimetypeicons-wgenial.rhcloud.com/.xls?size=64

http://mimetypeicons-wgenial.rhcloud.com/file.js

Using a default

http://mimetypeicons-wgenial.rhcloud.com/notfound?size=32&default=xml
